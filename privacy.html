Privacy Policy

Starry takes your privacy very seriously. We only store data that is necessary to the operation of Starry, like user IDs, guild IDs, role IDs, and channel IDs. Starry accesses more extensive data on users, roles, guilds, and channels when certain commands are run, but it is not stored.

The data Starry collects is stored only on the secure, password- and private-key protected servers that Starry is run on, and except for some command arguments, is never sent anywhere.

If you would like your data to be removed, just do 's!
mydata deleteme'. Commands will be coming soon to request a copy of your data, and list what is stored about you.

If you have any questions, DM CraziiAce#3062 on Discord.
